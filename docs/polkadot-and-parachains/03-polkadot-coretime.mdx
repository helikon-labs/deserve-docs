# Polkadot Coretime

Polkadot Coretime is a system parachain on Polkadot that manages the allocation and sale of coretime. Coretime is the
computational resource parachains use to produce blocks and get their state transitions validated by Polkadot's relay chain.

## Archive RPC endpoints

All endpoints support both `wss://` and `https://` protocols.
