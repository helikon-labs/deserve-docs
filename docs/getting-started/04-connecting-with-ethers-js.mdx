# Connecting with ethers.js

[ethers.js](https://docs.ethers.org) is a widely used JS/TS library for interacting with Ethereum-compatible chains. DeServe provides Ethereum-compatible RPC endpoints for Polkadot Asset Hub via the [Revive](https://github.com/paritytech/revive) pallet.

## Installation

```bash
npm install ethers
```

## Connecting to the geo-steered endpoint

<Tabs groupId="lang" defaultValue="js">
  <TabItem value="js" label="JavaScript">
    ```javascript
    import { ethers } from 'ethers';

    const provider = new ethers.JsonRpcProvider('https://asset-hub.polkadot.eth-rpc.deserve.network');

    const blockNumber = await provider.getBlockNumber();
    console.log(`Current block number: ${blockNumber}`);
    ```

  </TabItem>
  <TabItem value="ts" label="TypeScript">
    ```typescript
    import { ethers } from 'ethers';

    const provider = new ethers.JsonRpcProvider(
        'https://asset-hub.polkadot.eth-rpc.deserve.network',
    );

    const blockNumber = await provider.getBlockNumber();
    console.log(`Current block number: ${blockNumber}`);
    ```

  </TabItem>
</Tabs>

## Connecting with viem

[viem](https://viem.sh) is a **modern TypeScript alternative** to ethers.js:

```bash
npm install viem
```

<Tabs groupId="lang" defaultValue="js">
  <TabItem value="js" label="JavaScript">
    ```javascript
    import { createPublicClient, http } from 'viem';

    const client = createPublicClient({
        transport: http('https://asset-hub.polkadot.eth-rpc.deserve.network'),
    });

    const blockNumber = await client.getBlockNumber();
    console.log(`Current block number: ${blockNumber}`);
    ```

  </TabItem>
  <TabItem value="ts" label="TypeScript">
    ```typescript
    import { createPublicClient, http, PublicClient } from 'viem';

    const client: PublicClient = createPublicClient({
        transport: http('https://asset-hub.polkadot.eth-rpc.deserve.network'),
    });

    const blockNumber: bigint = await client.getBlockNumber();
    console.log(`Current block number: ${blockNumber}`);
    ```

  </TabItem>
</Tabs>
